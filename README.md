# Teman Mabar

## Project Description

Teman Mabar is a web application built with Laravel that helps gamers find teammates, create teams, organize competitions, and manage game-related orders and payments. The project includes user profiles, forums for discussions, team and competition management, and integrations for file storage and payments.

## Features

-   User registration, profiles, and authentication
-   Game and genre management
-   Team creation and membership management
-   Competition creation, scheduling, and participation
-   Forum with questions and comments
-   Order and payment processing
-   File storage integration (Firebase Storage service present)
-   Validation rules and custom request handling

## Technology Stack

-   PHP 8+ and Laravel framework
-   Composer for PHP dependency management
-   MySQL / relational database (migrations provided)
-   Vite, Tailwind CSS, and PostCSS for frontend tooling
-   Firebase Storage integration
-   PHPUnit for testing
